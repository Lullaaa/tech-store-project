# 🛒 Tech Store CLI Project

## 📌 Overview
This is a simple **Python Object-Oriented Programming (OOP) project** that simulates a basic store system with employees and customers.

I built this project after learning the fundamentals of **OOP in Python**, including classes, objects, methods, and basic program structure. The goal was to apply these concepts to a real-world interactive project.

---

## 🚀 Features

### 👨‍💼 Employee Features
*   **Secure Login:** Password-protected access for administrative tasks.
*   **Inventory Management:** Easily add or remove products from the store.
*   **Product Overview:** View all current inventory in a structured format.

### 🛍️ Customer Features
*   **Browsing:** View all available products in the store.
*   **Shopping Cart:** Add or remove items dynamically.
*   **Checkout:** Seamlessly calculate total prices for purchase.

---

## 🧠 Concepts Practiced
*   **Object-Oriented Programming (OOP):** Implementation of classes and objects.
*   **Encapsulation:** Managing data and behavior within logical units.
*   **Data Structures:** Using dictionaries for data storage.
*   **Control Flow:** Utilizing loops and conditionals for program logic.
*   **CLI Interaction:** Building a user-friendly Command Line Interface.
